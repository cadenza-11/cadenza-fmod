# Cadenza: FMOD Studio Repo
This repository contains the .fspro (FMOD Studio Project) project file used to curate and build the audio banks for the Cadenza Unity game.

## Installation
1. Clone this repository to your local filesystem. [*(how to clone?)*](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository)
1. Create an account at [fmod.com](https://www.fmod.com/profile/register) or login to an existing account.
2. Download **FMOD for Unity v2.03** package via the [FMOD page](https://www.fmod.com/download#fmodforunity) or the [Unity Asset Store](https://assetstore.unity.com/packages/tools/audio/fmod-for-unity-2-03-311497?srsltid=AfmBOopVRP83-mfgHkv2b-ufkIAiZJLk7SFx9tqi1PPboqz1-S6u54NS)
1. Download **FMOD Studio v2.03** for your operating system via the [FMOD page](https://www.fmod.com/download#fmodstudio).
1. Open **FMOD Studio v2.03**
1. Click **Open Project** and select the cloned *cadenza-fmod.fspro* file.

> Note: FMOD requires that you use the same major version for FMOD Studio, FMOD for Unity, and the .fspro file. This project uses major version v2.03. Please ensure you have downloaded the correct version of these products in order to avoid errors.
